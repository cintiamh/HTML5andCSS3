# Basic HTML Structure

## Creating your first web page

HTML5 Doctype:

```html
<!DOCTYPE html>
```

Basic structure:

```HTML5
<!DOCTYPE>
<html>
  <head></head>
  <body></body>
</html>
```

JavaScript `<script>` are right before the close `</body>`. Due to slow loading for script files.
